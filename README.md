## Инструкция по установке для студентов

#### После скачивания и открытия в VSCode пишем команду
### python -m venv venv
#### автивация в Windows
### .\venv\Scripts\activate
#### автивация в IOS
### source venv/bin/activate
#### pip install -r requirements.txt
#### python manage.py migrate
#### python manage.py createsuperuser
#### python manage.py runserver
### открываем локальный http://127.0.0.1:8000/
